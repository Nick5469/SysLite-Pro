## 简介
SysLite是一款免费的Windows系统定制工具。

### 特点
 - 系统appx、系统组件移除
 - 系统优化
 - 系统更新管理
 - 系统运行库配置
 - 系统服务管理
 - 系统全方位设置
 - 支持官方插件与第三方插件
 - 详细的系统定制教程
 - 软件支持跨平台使用（未实现）
